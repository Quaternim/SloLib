The SloLib Basics Page, Like a Tutorial

This page will explain the basics of SloLib and teach you how to select, hide and show HTML elements.



First, you will need to be able to select an element from your HTML. To do so you will need to type

$( )

then in the brackets you want to put either ↓

$("button.buttonclass")

for the element button with the class of buttonclass ↑ , you could also type ↓

$("p#paragraphid")

for p element with id of paragraphid ↑ or you could type ↓

const title = document.getElementById('titleid')
$(title)

to get the element with the id of titleid ↑
