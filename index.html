What Is SloLib??

SloLib is a lightweight and feature-full JavaScript library. It makes things like DOM manipulation, animation etc very simple.
It is very similar to jQuery and designed so jQuery code should work in SloLib.

A Few Examples of SloLib

This example shows how to hide an element in different ways using SloLib

$("button.hello").hide()

This ↑ would hide a button with the class hello

const var1 = document.getElementById('para')
$(var1).hide()

This ↑ would hide the element with the id of para, you could also do ↓

$(p#para).hide()

You can also use .show instead of .hide if the element is already hidden.
